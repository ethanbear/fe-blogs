# 代数

## 历史

- 伽利略
- 笛卡尔坐标系

## 基础

**抽象**

现实中的实物抽象化，抽象化实物符号化。

符号化的实物之间的联系使用代数的方式表达。
如
```
F(Force) 力 
a(Acceleration)加速度
m(Mass)质量

F = a * m 
```
>现实中的事物都可以抽象化，但某些抽象化的事物却不能实物化。比如一些凭空的想法，只能通过符号或文字表达。

**笛卡尔坐标系**

在代数与几何之间建立强大联系的发明者。
代数等式,表达了y和x之间的关系
```
y= 2x -1
```
我们可以可视化表达出这种关系，为了这个目标他发明了这个坐标系，x,y坐标系。他让抽象的符号代数时间和几何联系了起来，使它更具可视化。

**变量**

某一类值的符号化的表达。

**合并同类项**

2x+3x = 5x
