### 创建优于复用

编写样式时，优先使用新建classname新增样式属性替换直接使用原有样式追加样式方式。

通过新建classname和原有class组合使用，实现组件或文档样式，避免使用样式继承性及覆盖性方式。便于样式复用及解耦。
